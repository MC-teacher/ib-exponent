Unit 01 — Exponents

거듭제곱과 거듭제곱근

IB Core 1A · Representation: 수를 다르게 표현하는 방법

📐 Concept: Representation 🔗 IB Core 1A
📖 거듭제곱근의 정의

실수 \(a\)와 자연수 \(n\ (n\geq2)\)에 대하여 \(x^n = a\)의 근 \(x\)를 \(a\)의 \(n\)제곱근이라 한다. \(a\)의 제곱근, 세제곱근, 네제곱근, …을 통틀어 \(a\)의 거듭제곱근이라 한다.

\(a > 0\)\(a = 0\)\(a < 0\)
\(n\)이 짝수\(\sqrt[n]{a},\ -\sqrt[n]{a}\) 두 개0 하나없다
\(n\)이 홀수\(\sqrt[n]{a}\) 하나0 하나\(\sqrt[n]{a}\) 하나
🎓 Theory of Knowledge

음수의 짝수제곱근이 실수 범위에서 존재하지 않는다. 복소수로 확장하면 존재하게 되는데 — 이는 수학적 진리가 우리가 선택한 수 체계에 의존한다는 것을 의미하는가? 수학의 진리는 절대적인가, 조건적인가?

교과서 예제 1  (p. 13)

다음 거듭제곱근 중에서 실수인 것을 모두 구하시오.

(1)
\(-8\)의 세제곱근
(2)
\(81\)의 네제곱근
(1) — −8의 세제곱근

\(x^3 = -8\)로 놓으면 \(x^3 + 8 = 0\), 즉 \((x+2)(x^2-2x+4)=0\)

\(x^2-2x+4=0\)의 판별식 \(D = 4-16 < 0\) → 실근 없음

따라서 \(x+2=0\)에서 실수인 세제곱근: \(x = -2\)

(2) — 81의 네제곱근

\(x^4 = 81\)로 놓으면 \((x-3)(x+3)(x^2+9)=0\)

\(x^2+9=0\)은 실근 없음 → 실수인 네제곱근: \(x = \pm 3\)

答   (1) −2  |  (2) −3, 3

📐 거듭제곱근의 성질 (p. 16)

\(a>0,\ b>0\)이고 \(m,\ n\)이 2 이상인 자연수일 때,

①  \(\sqrt[n]{a}\cdot\sqrt[n]{b} = \sqrt[n]{ab}\)
②  \(\dfrac{\sqrt[n]{a}}{\sqrt[n]{b}} = \sqrt[n]{\dfrac{a}{b}}\)
③  \((\sqrt[n]{a})^m = \sqrt[n]{a^m}\)
④  \(\sqrt[m]{\sqrt[n]{a}} = \sqrt[mn]{a}\)

📐 유리수인 지수 (p. 21~22)

\(a>0\)이고 \(m,\ n\ (n\geq2)\)이 정수일 때,

\(a^{\frac{1}{n}} = \sqrt[n]{a}\)
\(a^{\frac{m}{n}} = \sqrt[n]{a^m} = (\sqrt[n]{a})^m\)

지수가 유리수일 때도 지수법칙 성립: \(a^r a^s = a^{r+s},\ (a^r)^s = a^{rs}\) 등

🎹 음악+ 문제 8   (p. 23) 융합 STEM

피아노 건반의 진동수는 건반의 상대 위치에 따라 유리수 지수로 나타낼 수 있다. 건반 ①의 진동수를 \(H\)라 하면 건반 ⑧의 진동수는

\(2^{\dfrac{n-1}{12}} \times H\)

라고 한다. 건반 ⑩의 진동수는 건반 ④의 진동수의 몇 배인지 구하시오.

건반 ④ (\(n=4\))

\(2^{\frac{3}{12}} \times H = 2^{\frac{1}{4}} \times H\)

건반 ⑩ (\(n=10\))

\(2^{\frac{9}{12}} \times H = 2^{\frac{3}{4}} \times H\)

배율 계산

\(\dfrac{2^{\frac{3}{4}} H}{2^{\frac{1}{4}} H} = 2^{\frac{3}{4}-\frac{1}{4}} = 2^{\frac{1}{2}} = \sqrt{2}\)

答   \(\sqrt{2}\) 배

04   집합과 거듭제곱근 ★ 기출
기출 ▶ 22054-0004  |  4점

집합 \(X = \{2,\,3,\,4,\,5,\,6\}\)에 대하여 집합 \(Y\)를

\(Y = \{(a,\,b) \mid a \in X \text{ 이고 } b \in X\}\)

라 하자. 집합 \(Y\)의 원소 \((a,\,b)\) 중에서 \(a\)의 \(b\)제곱근 중 실수인 것의 개수가 2인 원소의 개수는 \(m\)이고, \(-a\)의 \(b\)제곱근 중 실수인 것의 개수가 1인 원소의 개수는 \(n\)이다.

\(m + n\)의 값을 구하시오.

핵심 원리 — 교과서 p.15 표

\(n\)이 짝수이고 \(a > 0\) → 실수인 \(n\)제곱근 2개 (\(\sqrt[n]{a},\,-\sqrt[n]{a}\))

\(n\)이 홀수 → \(a\)의 부호에 무관하게 항상 1개

\(n\)이 짝수이고 \(a < 0\) → 실수인 \(n\)제곱근 0개

m 계산 — \(a\)의 \(b\)제곱근 중 실수가 2개인 \((a,b)\)

조건: \(b\)가 짝수이고 \(a > 0\)

\(X = \{2,3,4,5,6\}\)에서 \(a\)는 모두 양수 → \(a\) 선택 5가지

\(b \in \{2,\,4,\,6\}\) (짝수) → 3가지

\(\therefore\; m = 5 \times 3 = \mathbf{15}\)

n 계산 — \(-a\)의 \(b\)제곱근 중 실수가 1개인 \((a,b)\)

\(a \in X\)이므로 \(-a < 0\)

\(-a < 0\)일 때 실수인 \(b\)제곱근이 1개 → \(b\)가 홀수

\(b \in \{3,\,5\}\) (홀수) → 2가지, \(a\) 선택 5가지

\(\therefore\; n = 5 \times 2 = \mathbf{10}\)

答   \(m + n = 15 + 10 = 25\)
📌 IB 연결 — Representation: 거듭제곱근의 실수 개수를 n의 홀짝성 + a의 부호로 체계화하는 것이 핵심. 같은 수라도 표현 방식(제곱근의 차수)에 따라 구조가 달라진다.
07   표준 ⭐ (p. 38) 수능 유형

\(\sqrt{2} \times \sqrt[3]{2} \times \left(\sqrt[3]{2}\right)^{-\frac{9}{2}} = 2^k\)를 만족시키는 실수 \(k\)의 값을 구하시오.

유리수 지수로 변환

\(\sqrt{2} = 2^{\frac{1}{2}}\),  \(\sqrt[3]{2} = 2^{\frac{1}{3}}\),  \(\left(2^{\frac{1}{3}}\right)^{-\frac{9}{2}} = 2^{-\frac{3}{2}}\)

지수끼리 덧셈

\(k = \dfrac{1}{2} + \dfrac{1}{3} - \dfrac{3}{2} = \dfrac{3+2-9}{6} = -\dfrac{4}{6} = -\dfrac{2}{3}\)

答   \(k = -\dfrac{2}{3}\)
08   표준 ⭐ (p. 38) 수능 유형

양수 \(x\)에 대하여 \(x^{\frac{1}{2}} + x^{-\frac{1}{2}} = 3\)일 때, \(x^2 + x^{-2}\)의 값을 구하시오.

Step 1 — \(x+x^{-1}\) 구하기

\(\left(x^{\frac{1}{2}} + x^{-\frac{1}{2}}\right)^2 = x + 2 + x^{-1} = 9\)

∴ \(x + x^{-1} = 7\)

Step 2 — \(x^2+x^{-2}\) 구하기

\((x + x^{-1})^2 = x^2 + 2 + x^{-2} = 49\)

∴ \(x^2 + x^{-2} = 47\)

答   47
09   표준 ⭐ (p. 38) 수능 유형

실수 \(a\)에 대하여 \(\dfrac{2^a - 2^{-a}}{2^a + 2^{-a}} = \dfrac{1}{2}\)일 때, \(4^a + 4^{-a}\)의 값을 구하시오.

교차 곱셈

\(2(2^a - 2^{-a}) = 2^a + 2^{-a}\)

\(2 \cdot 2^a - 2 \cdot 2^{-a} = 2^a + 2^{-a}\), 즉 \(2^a = 3 \cdot 2^{-a}\)

\(2^{2a} = 3\), 따라서 \(4^a = 3\)

최종 계산

\(4^a + 4^{-a} = 3 + \dfrac{1}{3} = \dfrac{10}{3}\)

答   \(\dfrac{10}{3}\)
15   발전 ⭐⭐ (p. 39) 수능 고난도

실수 \(x\)에 대하여 \(x\)의 세제곱근 중에서 실수인 것의 개수를 \(f(x)\), \(x\)의 네제곱근 중에서 실수인 것의 개수를 \(g(x)\)라 하자.

\(|a| \leq 5\)와 \(|b| \leq 5\)를 만족시키는 두 정수 \(a\)와 \(b\)에 대하여 \(f(a)g(b) = 2\)를 만족시키는 순서쌍 \((a,b)\)의 개수를 구하시오.

\(f(x)\) 분석 — 세제곱근 (n이 홀수)

모든 실수 \(x\)에 대해 \(f(x)=1\) (홀수 제곱근은 항상 실수 1개)

\(g(x)\) 분석 — 네제곱근 (n이 짝수)

\(x > 0\): \(g(x) = 2\),  \(x = 0\): \(g(x) = 1\),  \(x < 0\): \(g(x) = 0\)

조건 분석

\(f(a) = 1\)이므로 \(g(b) = 2\) 필요, 즉 \(b > 0\)

\(|a| \leq 5\): \(a \in \{-5,\ldots,5\}\) → 11가지

\(|b| \leq 5,\ b>0\): \(b \in \{1,2,3,4,5\}\) → 5가지

총 \(11 \times 5 = 55\)

答   55
Unit 03 — Logarithms

로그

IB Core 1C · Representation — 지수의 역연산

📐 Concept: Representation 🔁 지수 ↔ 로그 역함수
📖 로그의 정의 (p. 26)

\(a>0,\ a\neq1\)일 때, 양수 \(N\)에 대하여 \(a^x = N\)을 만족시키는 실수 \(x\)는 오직 하나 존재한다. 이 \(x\)를

\(a^x = N \Longleftrightarrow x = \log_a N\)

\(a\)를 밑, \(N\)을 진수라 한다. 밑의 조건: \(a>0,\ a\neq1\). 진수의 조건: \(N>0\).

📐 로그의 성질 (p. 28)

\(a>0,\ a\neq1,\ M>0,\ N>0\)일 때,

①  \(\log_a 1 = 0, \quad \log_a a = 1\)
②  \(\log_a MN = \log_a M + \log_a N\)
③  \(\log_a \dfrac{M}{N} = \log_a M - \log_a N\)
④  \(\log_a M^k = k\log_a M\)
🎓 ToK — 로그의 역사

스코틀랜드 수학자 네이피어(Napier, 1550~1617)는 큰 수의 곱셈을 덧셈으로 바꾸는 로그표를 만들었다. 라플라스(Laplace)는 "로그는 천문학자의 수명을 배로 늘려주었다"고 했다. 수학적 도구의 발명은 단순한 편의인가, 아니면 새로운 진리의 발견인가?

■ 문제 3   (p. 27) 기본

다음 값을 구하시오.

(1)
\(\log_2 16\)
(2)
\(\log_3 \dfrac{1}{81}\)
(3)
\(\log_{\frac{1}{7}} 49\)
(4)
\(\log_4 \sqrt{64}\)

(1) \(\log_2 16 = \log_2 2^4 = 4\)

(2) \(\log_3 \dfrac{1}{81} = \log_3 3^{-4} = -4\)

(3) \(\log_{\frac{1}{7}} 49 = \log_{\frac{1}{7}} \left(\dfrac{1}{7}\right)^{-2} = -2\)

(4) \(\log_4 \sqrt{64} = \log_4 4^{\frac{3}{2}} = \dfrac{3}{2}\)

\(2\log_2\sqrt{3} - \log_2 6 + 6\log_2\sqrt{2}\)

🔄 로그의 밑의 변환 (p. 30)

\(a>0,\ a\neq1,\ b>0,\ c>0,\ c\neq1\)일 때,

\(\log_a b = \dfrac{\log_c b}{\log_c a}\)

특히 \((\log_a b)(\log_b a) = 1\),   \(\log_{a^m} b^n = \dfrac{n}{m}\log_a b\)

■ 문제 8   (p. 30) 표준

1이 아닌 두 양수 \(a\)와 \(b\)에 대하여 다음 등식이 성립함을 보이시오.

(1)
\((\log_a b)(\log_b a) = 1\)
(2)
\(\log_{a^m} b^n = \dfrac{n}{m}\log_a b\)  (\(m \neq 0\))
(1) 밑의 변환 적용

\(\log_a b = \dfrac{\log b}{\log a}\),  \(\log_b a = \dfrac{\log a}{\log b}\)

\((\log_a b)(\log_b a) = \dfrac{\log b}{\log a} \cdot \dfrac{\log a}{\log b} = 1\)  ■

(2) 밑의 변환 적용

\(\log_{a^m} b^n = \dfrac{\log b^n}{\log a^m} = \dfrac{n\log b}{m\log a} = \dfrac{n}{m}\cdot\dfrac{\log b}{\log a} = \dfrac{n}{m}\log_a b\)  ■

교과서 37~39쪽

문항 예시   함수방정식과 로그 ★ 기출
기출 함수방정식 유형  |  4점

양의 실수 전체의 집합에서 정의된 함수 \(f(x)\)가 임의의 양수 \(x\), \(y\)에 대하여

\(f(xy) = f(x) + f(y)\)

를 만족하고 \(f(2) = 1\)일 때, \(f(16)\)의 값을 구하시오.

구조 분석

조건 \(f(xy)=f(x)+f(y)\)는 교과서 p.28 로그의 성질 ②

\(\log_a MN = \log_a M + \log_a N\)과 완전히 동일한 구조.

밑 결정

\(f(x) = \log_a x\)로 놓으면 \(f(2) = \log_a 2 = 1\)

\(\therefore\; a = 2\), 즉 \(f(x) = \log_2 x\)

계산

\(f(16) = \log_2 16 = \log_2 2^4 = 4\)

검증 — 함수방정식으로 직접 확인

\(f(16) = f(2\cdot2\cdot2\cdot2)\)

\(= f(2)+f(2)+f(2)+f(2) = 1+1+1+1 = 4\) ✓

答   4
📌 IB ToK 연결: \(f(xy)=f(x)+f(y)\)는 곱셈을 덧셈으로 변환하는 구조 — 이것이 바로 네이피어가 로그를 고안한 이유이자 항해사들이 로그표를 사용했던 핵심이다. Representation: 연산의 복잡도를 낮추는 더 좋은 표현법의 발명.

11   서술형 ✏️ (p. 38) 서술형

다음을 모두 만족시키는 두 양수 \(a\)와 \(b\)에 대하여 \(a^2 + b^2\)의 값을 구하는 풀이 과정과 답을 쓰시오.

㈎  \(\log_2 a + \log_2 b = 1\)

㈏  \(\log_2\!\left(1+\dfrac{1}{a}\right) + \log_2\!\left(1+\dfrac{1}{b}\right) = 3\)

조건 ㈎

\(\log_2 ab = 1 \Rightarrow ab = 2\)

조건 ㈏

\(\log_2\!\dfrac{(a+1)(b+1)}{ab} = 3 \Rightarrow (a+1)(b+1) = 8ab = 16\)

\(ab + a + b + 1 = 16 \Rightarrow 2 + a + b + 1 = 16 \Rightarrow a + b = 13\)

최종 계산

\(a^2 + b^2 = (a+b)^2 - 2ab = 169 - 4 = 165\)

答   165

18   발전 ⭐⭐ 서술형 (p. 39) 수능 고난도

1보다 큰 두 실수 \(a\)와 \(b\)에 대하여

\(\log_{\sqrt{2}} a = \log_4 ab\)

가 성립할 때, \(\log_a b\)의 값을 구하는 풀이 과정과 답을 쓰시오.

밑을 2로 통일

\(\log_{\sqrt{2}} a = \dfrac{\log_2 a}{\log_2 \sqrt{2}} = \dfrac{\log_2 a}{\frac{1}{2}} = 2\log_2 a\)

\(\log_4 ab = \dfrac{\log_2 ab}{2} = \dfrac{\log_2 a + \log_2 b}{2}\)

등식 풀기

\(2\log_2 a = \dfrac{\log_2 a + \log_2 b}{2}\)

\(4\log_2 a = \log_2 a + \log_2 b\)

\(\log_2 b = 3\log_2 a\), 따라서 \(\log_a b = 3\)

答   3

📐 탐구 활동 1 — 역설계: m+n = 30이 되는 집합 X 설계하기
기출 문제(22054-0004)에서 \(X=\{2,3,4,5,6\}\)일 때 \(m+n=25\)였다.
\(m+n=30\)이 되도록 집합 \(X\)를 설계할 수 있을까?
공식 도출

집합 \(X\)에서 양수의 개수를 \(p\), 집합의 크기를 \(|X|\)라 하면:

\(m = p \times (\text{짝수 } b\text{의 수}),\quad n = p \times (\text{홀수 } b\text{의 수})\)
\(m + n = p \times |X|\)

짝수든 홀수든 양수 \(a\)는 항상 한 쪽에 카운트되므로, \(b\)의 홀짝 분포와 무관하게 성립한다.

Step 1 — b의 조건 재확인

집합 \(X\)의 원소는 \(a\)로도 쓰이고 \(b\)(거듭제곱근의 차수)로도 쓰인다.
\(b\)는 반드시 2 이상의 자연수여야 하므로, 집합 \(X\)의 원소는 모두 자연수.

Step 2 — 공식 단순화

모든 원소가 양수(자연수)이므로 \(p = |X|\).

\(m + n = p \times |X| = |X|^2\)

가능한 \(m+n\) 값은 반드시 완전제곱수:

\(|X|=2\)
4
\(|X|=3\)
9
\(|X|=5\)
25
\(|X|=6\)
36
Step 3 — 30은 완전제곱수인가?

\(\sqrt{30} \approx 5.47\cdots\) — 자연수가 아니다.

결론: 조건을 지키는 자연수 집합으로 \(m+n=30\)을 만드는 것은 불가능하다.
Step 4 — 흔한 오류: X = {2, 3, 4, 5, 6, −2} 를 시도하면?

표면상: 양수 5개, \(|X|=6\) → \(5\times6=30\) ✓ 처럼 보이지만…

\(b=-2\)는 "−2제곱근"으로 정의 자체가 성립하지 않는다.
집합의 원소가 \(b\)로 쓰일 때는 반드시 자연수여야 하므로 이 설계는 무효.

∴ m+n이 될 수 있는 값 = 완전제곱수만 가능  |  30은 불가능
ToK 연결: "답이 없다"는 것을 증명하는 것도 수학적 지식이다. 물리학은 "없을 것 같다"고 말하지만, 수학은 "절대로 없다"고 말할 수 있다.

질문 1 그래프 해석

\(n=4,\; a=-1\)일 때 교점이 없는 이유를 그래프로 설명하시오.

그래프 관찰

\(y = x^4\)은 \(n\)이 짝수이므로 U자형(아래로 볼록) 곡선으로, 모든 \(x\)에 대해 \(y \geq 0\)이다.

\(y = -1\)은 \(x\)축 아래에 있는 수평선이다.

교점이 없는 이유

\(x^4 \geq 0\)이므로 \(x^4 = -1\)을 만족하는 실수 \(x\)는 존재하지 않는다.

\(y=x^4\)의 최솟값은 0이고, \(y=-1\)은 그보다 아래에 있으므로 두 그래프는 절대 만나지 않는다.

💡 교과서 표에서 "\(n\)이 짝수, \(a < 0\)이면 실수인 근이 없다"는 것의 그래프적 의미가 바로 이것이다.
질문 2 그래프 해석

\(n=3,\; a=-8\)의 교점이 항상 1개인 것은 어떤 그래프 특성 때문인가?

그래프 관찰

\(y = x^3\)은 \(n\)이 홀수이므로 원점 대칭(점대칭) 곡선이다.

\(x\)가 \(-\infty \to +\infty\)로 갈 때 \(y\)도 \(-\infty \to +\infty\)로 단조증가한다.

교점이 항상 1개인 이유

단조증가하는 연속함수는 모든 수평선 \(y = a\)와 정확히 1번 만난다.

\(x^3 = -8\)의 실수 해는 \(x = -2\) 단 하나.

짝수 차수와의 비교

\(y = x^2\)은 U자 대칭 → \(a > 0\): 교점 2개, \(a = 0\): 1개, \(a < 0\): 0개.

홀수 차수 함수는 y축 대칭이 없고 점대칭만 있으므로 수평선과 항상 정확히 1개 교점.

💡 "단조성 + 연속성"의 조합이 교점 1개를 보장한다. 이는 중간값 정리(IVT)와 연결되는 개념이기도 하다.
🌐 탐구 활동 2 — 수학의 대칭 vs 자연의 대칭

"수학은 왜 이 패턴을 예외 없이 따르는가? 자연에서 대칭이 깨지는 사례와 비교할 때, 수학의 대칭은 더 신뢰할 수 있는가?"

수학의 대칭 — 정의에서 오는 필연

\((-x)^n = x^n\) (\(n\)이 짝수)은 실험으로 확인하는 것이 아니라,
음수×음수=양수라는 곱셈의 정의로부터 논리적으로 따라온다.

공리 → 논리 → 결론의 사슬.
전제가 참이면 결론은 반드시 참.
반례는 원리상 불가능하다.

자연의 대칭 — 관찰에서 오는 패턴

실험으로 확인된 패턴.
더 정밀한 실험에서 깨질 수 있다.

CP 대칭 붕괴 (1964, Cronin & Fitch):
물질·반물질 완전 대칭 예측 → 중성 케이온 붕괴에서 비대칭 발견. 노벨상 수상.

패리티(P) 붕괴 (1957, Wu 실험):
약한 핵력은 좌우 대칭이라 예상 → 전자가 한 방향으로만 방출.

항목 수학의 대칭 자연의 대칭
근거 공리 + 논리적 연역 실험 + 귀납적 일반화
반례 가능성 없음 (정의상 불가) 언제든 가능
깨지는 방식 공리를 바꾸면 다른 수학이 됨
예: 비유클리드 기하학		 더 정밀한 실험으로 이론 수정
신뢰의 종류 논리적 확실성 경험적 개연성
💡 ToK 핵심 논지

수학이 예외 없이 대칭을 따르는 것은, 수학이 현실을 묘사하는 언어가 아니라 현실과 무관하게 자기 완결적인 논리 체계이기 때문이다.

"수학은 우리가 무엇에 대해 말하는지 모르고, 우리가 말하는 것이 참인지도 모른다."
— Bertrand Russell

역설적으로, 수학이 물리학보다 더 확실한 이유는 수학이 현실에 덜 관심을 갖기 때문일 수 있다.
